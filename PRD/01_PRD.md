# Design Auto — Production Upgrade PRD

## 1. 왜 만드는가

현재 Design Auto는 "Claude가 HTML을 짜주고 Playwright로 이미지 캡처"하는 **1회용 생성기**다.
실무자(마케터·디자이너·운영자)가 실제 업무에 쓰려면 다음 네 가지 문제가 치명적이다.

1. **편집 불가** — 글자 한 줄 바꾸려고 전체 재생성. 비용·시간 낭비.
2. **휘발성** — 페이지 새로고침하면 결과물이 사라짐. 과거 작업 재사용 불가.
3. **브랜드 불일치** — 매번 색·폰트·로고를 수동 입력. 같은 프로젝트 내 결과물끼리도 톤이 다름.
4. **품질 편차** — 같은 입력에도 결과 품질이 들쭉날쭉. 실패 시 재시도 가드 없음.

목표: **실무자가 "한 번 생성 → 다듬어서 바로 업로드"까지 갈 수 있는 수준**으로 올린다.

## 2. 누가 쓰는가

- **주 사용자**: 혼자 혹은 팀 내부 (2~10명 규모)
- **업무 맥락**: 상세페이지·썸네일·카드뉴스·패키지 목업을 주 1~수십 회 뽑아야 하는 마케팅/커머스 운영자
- **기술 수준**: 코드 못 읽어도 사용 가능해야 함

## 3. 무엇을 만드는가 (핵심 기능)

### A. 결과물 편집 (Inline Editing) — 최우선
- 생성된 결과 위에서 **텍스트 클릭 → 바로 수정**
- **색·폰트·크기** 인스펙터에서 조정
- **이미지 영역 교체** (업로드 / AI 재생성 / URL)
- **블록 단위 재생성** ("이 슬라이드만 다시" / "이 문단만 다시")
- **Undo/Redo + 버전 스냅샷**

### B. 프로젝트·히스토리
- 모든 생성물은 **프로젝트**에 귀속 (기본 프로젝트 1개 자동 생성)
- 각 결과물은 **썸네일 + 메타데이터**로 리스트에서 탐색 가능
- **복제**("이거 기반으로 다시") · **즐겨찾기** · **검색**
- 버전: 편집 시 자동 스냅샷, 필요하면 롤백

### C. 브랜드 프리셋
- **로고 업로드 + 브랜드 컬러(1~5) + 폰트(헤드·본문) + 톤앤보이스 메모** 저장
- 생성 시 프리셋 선택만 하면 자동 주입
- 프리셋 여러 개 (브랜드 A / B / 이벤트용 등)

### D. 품질 가드
- 생성 후 **미리보기 → 확정** 플로우 (즉시 저장 대신 확정 버튼)
- Claude 응답 파싱 실패 시 **자동 1회 재시도**
- 레퍼런스 이미지 업로드 → **비전 모델로 스타일 추출 → 프롬프트 보강**
- 캡처 실패 시 에러 원인 노출 + 수동 재시도 버튼

### E. 운영 안정성 (최소선)
- **API 키 서버 보관** (현재 유지, 클라이언트 노출 금지 재확인)
- **기본 인증**: 팀용 단일 비밀번호 (Basic Auth) — 배포 URL 보호
- **요청 로그**: 누가·언제·얼마(토큰) 썼는지 파일/DB 기록
- **에러 로깅**: 실패 요청 추적 가능

## 4. 하지 않는 것 (Out of Scope, v1)

- 결제·요금제·구독
- 멀티테넌시(회사별 격리)
- 실시간 협업(동시 편집)
- 공개 공유 링크
- 모바일 앱
- 벡터 출력(SVG/AI) — PNG/JPG만

## 5. 성공 기준 (Definition of Done)

실무자 테스트 시나리오를 **중단 없이** 완주:

1. 브랜드 프리셋 1개 등록
2. 썸네일 3개 생성 (프리셋 선택)
3. 그중 1개 선택 → 제목 텍스트 수정 + 배경색 변경 + 이미지 영역만 재생성
4. 프로젝트에 저장 → 다음 날 다시 열어 편집 가능
5. PNG 다운로드 → SNS 업로드

위 5단계가 **재생성 루프 없이, 페이지 새로고침에도 살아남고, 5분 이내**에 끝나야 한다.

## 6. 비기능 요구사항

| 항목 | 기준 |
|------|------|
| 응답 시간 | 생성 60초 이내, 편집 반영 1초 이내 |
| 동시 사용자 | 팀원 5명 동시 작업 무장애 |
| 데이터 보존 | 프로젝트·히스토리 영구 보존 (명시 삭제 전까지) |
| API 비용 | 월 사용량 대시보드 제공, 한도 설정 가능 |
| 보안 | API 키 서버 전용, 배포 URL Basic Auth |
