## 1. Minimal (미니멀형)

Front panel:
```
┌─────────────────┐
│   [logo]        │
│                 │
│                 │
│   Product Name  │
│   variant info  │
│                 │
│        net wt.  │
└─────────────────┘
```

Design rules:
- Lots of whitespace (60%+ of panel area)
- Typography-driven: product name is the hero element
- Muted, desaturated colors
- Minimal decoration: thin lines, subtle borders only
- Single accent color for key info (variant, weight)

Back panel:
- Clean grid layout for ingredients/nutrition
- Generous line spacing in legal text
- Barcode centered at bottom with ample clear space

Side panel:
- Single tagline or brand philosophy statement
- Minimal: brand name + website only

## 2. Illustration (일러스트형)

Front panel:
```
┌─────────────────┐
│ ╭─╮  [logo]    │
│ │♣│  ~~~ ~~~   │
│ ╰─╯  ~~~ ~~~   │
│  ☘️  Product    │
│  🌿  Name      │
│      variant    │
│        net wt.  │
└─────────────────┘
```

Design rules:
- Decorative patterns fill 40-50% of panel area
- Botanical, geometric, or abstract motifs in brand colors
- Hand-drawn or organic feel
- Multiple accent colors (brand main + sub + tints)
- Product name embedded within illustration composition
- Border or frame using illustrative elements

Back panel:
- Illustration border/frame continuing from front
- Ingredients in styled list (not plain table)
- Decorative dividers between sections

Side panel:
- Mini illustration or pattern strip
- Brand story in conversational tone
- Ingredient highlights with icons

## 3. Photo (포토형)

Front panel:
```
┌─────────────────┐
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
│▓▓ Product ▓▓▓▓▓▓│
│▓▓  Photo  ▓▓▓▓▓▓│
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
│   Product Name   │
│   variant info   │
│        net wt.   │
└─────────────────┘
```

Design rules:
- Product photography occupies 50-60% of front panel
- Photo area: placeholder with gradient if no actual image
- Clean bottom strip for text info (white or light bg)
- Professional, commercial feel
- Minimal text on photo area (brand logo overlay only)
- Sharp contrast between photo zone and text zone

Back panel:
- Structured grid: nutrition facts in standard table format
- Product usage photos (small thumbnails) if applicable
- Clean, corporate feel matching front

Side panel:
- Narrow product image strip or color block
- Key benefit in bold text
- Certification/quality marks
